Oral Chess

About

Oral Chess is a speech recognition extension which provides users with verbal control of peices and commands on Lichess.org. Users have the option to play blindfolded, hear audible responses, receive confirmation and real-time notifications. To contribute suggestions, report issues or for more information, feel free to reach us at OralChess@gmail.com.




Commands

Action Command
Propose a takeback Takeback
Resign Resign, Quit



Notes

Google Chrome

Unfortunately, the only browser currently supported is Google Chrome. As of now, other browsers do not support speech recognition.

Lichess

Lichess is a free online chess game server focused on realtime gameplay and ease of use. Visit Lichess.org to begin using Oral Chess.

Enabled

While enabled, users can customize preferences and provide verbal commands in game.

Blindfold

Blindfold hides peices in game.

Confirmation

Confirmation provides users with verification before executing verbal commands.

Notification

Notification provides users with real-time information.

Text To Speech

Text To Speech provides users with audible responses.
