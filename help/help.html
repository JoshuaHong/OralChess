Oral Chess

About

Oral Chess is a Google Chrome speech recognition extension which provides users with verbal control over peices and commands on Lichess.org. Users have the option to play blindfolded, hear audible responses, receive confirmation and real-time notifications. To contribute suggestions, report issues or for more information, feel free to reach us at OralChess@gmail.com.

Notes

Blindfold

Blindfold hides peices in game.

Confirmation

Confirmation provides users with verification before executing verbal commands.

Enabled

While enabled, users can customize preferences and provide verbal commands in game.
To provide verbal commands, join a game in Lichess and ensure that microphone is enabled in Google Chrome.

Google Chrome

The only browser currently supported is Google Chrome. As of now, other browsers do not support speech recognition.

Keyboard Input

Enable keyboard input to begin using Oral Chess. Keyboard input can be enabled in Lichess under "Preferences" > "Game behavior" > "Input moves with the keyboard".

Lichess

Lichess is a free online chess game server focused on realtime gameplay and ease of use. Visit Lichess.org to begin using Oral Chess.

Notification

Notification provides users with real-time information.

Text To Speech

Text To Speech provides users with audible responses. Headphones recommended.
